# flutter-demo
flutter 基础项目

## 功能
* 环境变量
* 日志
* 打包脚本
* 全局数据(provider)
* 路由(go_router)
* svg图标(flutter_svg)
* MVVM页面示例
* 深浅主题切换
* 全局样式
* 底导航
* AppBar
* 底部弹框 BottomSheet

* 自定义字体
* 网络请求
* 二维码扫描
* toast
* 时间日期选择器
* 模态弹框组件
* 国际化

# 安卓打包
运行 flutter build apk

# iOS打包
直接打ipa包，运行 flutter build ipa
打包后，需要使用 Transporter 上传。
