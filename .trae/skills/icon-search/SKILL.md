# 图标搜索技能

## 描述

该技能用于搜索和获取符合项目设计风格的SVG图标，并将其保存到assets/icon目录下。

## 项目图标风格规范

### 阿里巴巴国际站图标库风格特点
参考：https://www.iconfont.cn/collections/detail?cid=19238

| 规范项 | 要求 |
|--------|------|
| 尺寸 | 1024 × 1024 viewBox |
| 风格 | 扁平化、简洁线条 |
| 路径 | 单路径绘制，无渐变、阴影 |
| 颜色 | 单色，由代码动态渲染 |
| 变体 | 支持描边(line)和填充(fill)两种版本 |
| 圆角 | 统一的圆角处理 |
| 命名 | 小写字母+中划线，fill变体加-fill后缀 |

### 现有图标示例
- `home.svg` - 描边版本
- `home-fill.svg` - 填充版本
- `search.svg` - 描边版本
- `add.svg` - 描边版本

## 使用场景

当用户需要添加新图标时触发，例如：
- "帮我找一个用户图标"
- "需要一个购物车图标"
- "添加一个设置图标"

## 工作流程

### Step 1: 解析用户需求
理解用户需要的图标内容和类型（描边/填充）

### Step 2: 搜索优先级

按以下顺序搜索图标：

1. **优先搜索阿里巴巴国际站图标库**
   - 访问：https://www.iconfont.cn/collections/detail?cid=19238
   - 搜索相关图标
   - 下载SVG格式

2. **通用搜索iconfont.cn**
   - 如果在阿里巴巴国际站未找到
   - 在https://www.iconfont.cn/搜索风格相近的图标
   - 选择符合项目风格的图标下载

3. **自定义绘制**
   - 如果以上都未找到
   - 根据项目风格规范，手动绘制SVG图标
   - 确保符合1024×1024 viewBox和简洁线条风格

### Step 3: 图标处理
- 确保SVG符合项目风格规范
- 移除多余属性，保留纯路径
- 统一命名格式

### Step 4: 保存图标
- 存储到 `/Users/jay/Code/flutter-demo/assets/icon/` 目录
- 使用小写字母+中划线命名
- 填充版本添加 `-fill` 后缀

## 示例

**输入：**
```
帮我找一个购物车图标
```

**执行流程：**
1. 在阿里巴巴国际站图标库搜索"购物车"
2. 找到合适图标，下载SVG
3. 检查是否符合风格规范
4. 保存为 `assets/icon/shopping-cart.svg`

**输出：**
```
已找到并保存购物车图标：
- assets/icon/shopping-cart.svg
```

## 注意事项

1. **风格一致性** - 确保新图标与现有图标风格一致
2. **命名规范** - 使用小写字母和中划线，避免特殊字符
3. **格式检查** - 确保SVG文件有效且可在Flutter中使用
4. **文件大小** - 保持SVG简洁，移除不必要的元数据
